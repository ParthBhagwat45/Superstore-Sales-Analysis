# Superstore Sales Analysis

An end-to-end Data Analytics project that analyzes retail sales performance using **Python**, **Pandas**, and **Tableau** to uncover business insights and support data-driven decision-making.

---

## Project Overview

This project analyzes the Superstore retail dataset to evaluate sales performance, profitability, customer behavior, regional performance, product categories, and shipping operations. The data was cleaned and analyzed using Python, followed by the development of two interactive Tableau dashboards for business visualization and business decision support.

---

## Business Objective

The objective of this project was to:

- Analyze sales and profit trends over time.
- Evaluate category and sub-category performance.
- Identify high-performing and low-performing regions.
- Understand customer segment contribution.
- Analyze the impact of discounts on profitability.
- Evaluate shipping performance.
- Develop interactive dashboards to support business decision-making.

---

## Dataset

- **Dataset:** Sample Superstore Dataset
- **Source:** [Kaggle - Sample Superstore Dataset](https://www.kaggle.com/datasets/konstantinognev/sample-superstorecsv)
- **Records:** 9,994
- **Time Period:** 2014 – 2017

## Project Workflow

### Ask
- Understand the business problem and define project objectives.

### Prepare
- Collect the Superstore dataset from Kaggle.
- Review dataset structure and understand available variables.

### Process
- Clean and preprocess the dataset.
- Handle missing values.
- Create new features and calculated metrics.
- Validate data quality.

### Analyze
- Perform Exploratory Data Analysis (EDA).
- Calculate Key Performance Indicators (KPIs).
- Identify trends, patterns, and business opportunities.

### Share
- Develop two interactive Tableau dashboards to communicate insights.

### Act
- Provide business insights and actionable recommendations based on the analysis.
